Class ComponentInfoImpl<R>

提供一个组件信息的封装，包括：

  • computed name 列表、filters name 列表、root ANode
  • 从 TypeScript 来的还有 sourceFile、classDeclaration 等

注意：

  • 这里只是存数据，它的创建由具体 parser 负责
  • 这个工具类只为了方便实现具体的 ComponentInfo，不暴露作为接口
